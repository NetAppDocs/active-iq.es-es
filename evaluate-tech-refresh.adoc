= Determina si se recomienda una actualización del sistema para tu sistema de almacenamiento en Digital Advisor
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Si quieres identificar si se recomienda una actualización del sistema para tu contrato de Tech Support o hardware, puedes usar la opción de actualización del sistema.

Puedes acceder a esta función a través de la planificación del ciclo de vida (*Storage* > *Lifecycle planning*) o a través de Digital Advisor (*Dashboard > Planning widget > System Refresh*).

Para más información sobre esta función, consulta link:https://docs.netapp.com/us-en/bluexp-economic-efficiency/use/tech-refresh.html["Evaluar una actualización del sistema"] en la documentación de la consola.
